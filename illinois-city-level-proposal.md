# Illinois Laser Directory Proposal

## Executive Summary

This proposal includes Illinois candidates for the USA Laser Directory, researched using city/metro-level methodology with stricter contact extraction requirements.

## Target Cities Used

- **Chicago** (primary metro)
- **Naperville**
- **Aurora**
- **Springfield**
- **Peoria**
- **Rockford**
- **Lisle**
- **Deerfield**
- **Westchester**
- **Bensenville**
- **Joliet**

## Research Methodology

1. City/metro-level search for each target location
2. Categories: Engraver, Distributor, Sign Company
3. Stricter contact extraction order:
   - Visible homepage text
   - Header/footer contact info
   - mailto: links
   - Contact/about page
   - LinkedIn/Instagram links
   - Broader web search only if still missing
4. Prefer candidates with website + at least one contact channel
5. For distributors: prefer website + email + represented brand
